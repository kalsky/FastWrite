Quick Start

After installation FastWrite is ready to go.
All you have to do is sit down, stretch your HAND and start writing fast.

The default type mode after installation is SingleTap. In this mode you have to press once on each key to build your word (hello = 49556 ).

After you tap the word, if FastWrite didn't give you the right word, you can flip forward and backward through the matching words using the NEXT '*' and PREV '/' keys accordingly.

To change modes you can use the 'Num Lock' key or left click on the tray icon.

If you want a capital letters in your words you have to do the following:

After you write your word, press the '+' key, it will make the first letter uppercased. Press the '+' key again and it will make the whole word uppercased, press it for the third time and the whole word will return to lowercase.

Enjoy your FastWrite.


Type Modes

MultiTap
Tapping on one key will cycle through that key letters.
Use this mode if you cannot find your word in FastWrite dictionary (using the '*' key to go though the matching words in SingleTap or SingleTap+ modes ).
When you tap on a key, it will show you the first letter of that key (for instance 'g' when you press on '4'). Tap again on the same key and the letter will change to the second letter, and so on... Once you found the desired key, wait for the caret to move right (or move with the arrow keys by yourself), and tap the next letter.

SingleTap
In this mode you have to press once on each key to build your word.
All you have to do when you want to write a word is to tap once on each key that has the current letter on the word.
For example if you want to write 'hello', just tap '49556'.
If FastWrite didn't give you the word that you meant, cycle through the matching words using the NEXT key '*' and PREV key '/'. To accept the word and move on, press the SPACE '0' key.

SingleTap +
Like SingleTap but in this mode FastWrite will complete the word for you .
While you type the first letters of a word, FastWrite will show you the most commonly word that start with these letters.
For example , if you try to write 'possible', after 3 letters FastWrite will give you this word, when the 'pos' part is written and the 'sible' part is selected, like this: possible
If FastWrite found the word that you've meant, just press on the SPACE '0' key to accept the word, otherwise, keep tapping on the next letters of your word, or use the NEXT '*' and PREV '/' keys to find another possible match.
If FastWrite gave you a word that contains your word ('portable' when you wanted to write 'port'), you can use the NEXT and PREV keys to find your word, or you can use the BACK '-' key do delete the selected part, and then use the SPACE key to accept the word.

Numbers
In this mode the Numpad is used as if FastWrite doesn't work, and the Num Lock is ON.

Off
In this mode the Numpad is used as if FastWrite doesn't work, and the Num Lock is OFF.

Special Keys

Num Lock
Cycles through MultiTap/SingleTap/SingleTap+/Numbers/Off type modes.

Next ‘*' and Prev ‘/'
Cycles through the possible words, according to the letters that were tapped.
In MultiTap, cycling is done on the last key that was tapped.
In SingleTap, cycling is done on the possible words that match the keys that were tapped.
In SingleTap+, cycling is done on the possible words that match the keys that were tapped and on available words that starts with the matching keys that were tapped.
In Numbers and Off modes, these keys have the default usage (‘*' and ‘/').

Back ‘-‘
This key has the function of the Back Space.
Use it to delete words/letters if you had mistakes.
That way if you delete only part of the word, you can continue typing and FastWrite will find the word for you.
Note: in SingleTap+ mode you can use the ‘-‘ key to delete the completion that FastWrite suggested.

Capital Letters ‘+'
In MultiTap, after you found the desired letter, you can use the ‘+' key to make it uppercased or lowercased.
In SingleTap and SingleTap+, after you write your word, press the '+' key, it will make the first letter uppercased. Press the '+' key again and it will make the whole word uppercased, press it for the third time and the whole word will return to lowercase.

Space ‘0' / Enter
In SingleTap and SingleTap+, the ‘0' key has the function of the Space Bar.
Once you typed your word and you want to move on to the next word, use the ‘0' to approve the word you've just typed (the Enter key will approve the word and get down a line).
Note: in SingleTap+ you can use the ‘0' key if FastWrite completed the word for you and you want to move on to the next word.

Add ‘.'
If you find a word that is not in the FastWrite dictionary and you want to add it for future use, you may use the ‘.' Key.
First you will have to type the word (either in MultiTap or SingleTap+MultiTap or using the standard keyboard) and than while the caret is at the end of the word, press the ‘.' Key.
Note: an easier way to add words is through the settings option ‘Teach FastWrite your words'.

Sym '7'
If you need to type a symbol character, use the '7' key, and cycle through all the options using the Next and Prev keys.

Settings

Startup
 Start with Windows - Starts FastWrite on windows startup.
Check for updates on Startup - Each time FastWrite is started, it checks for updates availability.
Start Minimized - Opens FastWrite minimized.

Default Type Mode
Choose the type mode that will be active each time on the start-up of FastWrite, or choose to save the last used mode.

Functionality
 Fast Delete - If not selected, FastWrite will try to connect to words while you delete so that you may continue to write from a middle of a word. If you select this option, FastWrite will only know how to deal with the current word, and will delete previous words without connecting to them...
Auto add new words - While you type, after each Space hit, the words you just entered will be added to the dictionary if it is not exist.

Words Priority Database
The words priority database is the list of words that FastWrite knows and their priorities by which you get the words as you type (using SingleTap and SingleTap+). While you type the priority of the words is changing according to your selections and adapting the database to your use.
With the following settings you can change the priorities database:
Set to default – sets the priorities to their default state (like after installation)
Teach FastWrite your words – allows you to give FastWrite some text that contains your mostly used words. You may give some text from your daily emails, documents, chats, source code, etc.
The more words you give, the more accurate FastWrite will be for you.

Main Window Transparency
Move the slide bar to set the level of transparency for the main FastWrite window.
